# World Development Data Explorer - Shiny App

## Overview

The World Development Data Explorer is an interactive Shiny application designed to visualize and analyze key development indicators across countries globally. This app provides insights into various socio-economic and health metrics, allowing users to explore trends, compare countries, and understand the relationships between different indicators.

## Features

The app is structured into four main sections:

1. **Country Profile Dashboard**: 
   - Users can select a country to view its key development indicators like GDP, population growth, life expectancy, and more over time.

2. **Global Trends Visualization**: 
   - View global trends for selected indicators.
   - The app highlights trends of a chosen country, making it easy to compare it against the global backdrop.

## Data Source

The app utilizes the `world_development_data_imputed.csv` dataset, which comprises various indicators covering demographic, economic, and health aspects.

## Tools and Technologies

- **R and Shiny**: The app is built using R and the Shiny framework for interactive web applications.
- **ggplot2**: For creating sophisticated visualization charts.
- **dplyr**: Used for data manipulation and preparation.

## Getting Started

To run the app locally:

1. Clone this repository.
2. Open the R project in RStudio.
3. Run `app.R` to launch the application.
